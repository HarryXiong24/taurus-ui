# Introduction

A UI components built with React Hooks & Typescript.

🏠 [Homepage](https://github.com/HarryXiong24/Xbear-UI)

## Install

```sh
yarn install
```

## Usage

```sh
yarn dev
```

## Run tests

```sh
yarn test
```
